# OpenCLI 程序目录结构

本文档描述本仓库的 **目录布局（directory layout）** 与主要职责，便于快速定位代码与文档。生成时以源码树为准；构建产物与依赖目录不展开。

**排除路径（不列入树内）**：`node_modules/`、`dist/`、`extension/node_modules/`、`extension/dist/`、`docs/.vitepress/cache/`、`.git/`。

---

## 一、仓库根目录一览


| 类型       | 路径                                                                                         | 说明                             |
| -------- | ------------------------------------------------------------------------------------------ | ------------------------------ |
| 配置       | `package.json`、`package-lock.json`、`tsconfig.json`                                         | Node 包与 TypeScript 编译          |
| 说明       | `README.md`、`README.zh-CN.md`、`CHANGELOG.md`、`CONTRIBUTING.md`、`PRIVACY.md`、`TESTING.md` 等 | 项目说明与约定                        |
| CLI 说明   | `CLI-ONESHOT.md`、`CLI-EXPLORER.md`                                                         | 子命令/工具文档                       |
| 入口构建     | `scripts/`                                                                                 | 清理、复制 YAML、postinstall、站点测试脚本等 |
| 主源码      | `src/`                                                                                     | CLI 核心、浏览器桥、流水线、站点适配器          |
| 扩展       | `extension/`                                                                               | 浏览器扩展（独立子包）                    |
| 文档站      | `docs/`                                                                                    | VitePress 文档（中英、adapter 说明等）   |
| 测试       | `tests/`                                                                                   | smoke、e2e 等                    |
| 自动化      | `.github/`                                                                                 | Actions、PR 模板、Issue 模板         |
| Agent 技能 | `skills/`                                                                                  | opencli 相关 SKILL 与说明（非运行时）     |

| 其他       | `twitter/`                                                                                 | 仓库内示例/资源目录（非核心源码）              |

## 五、子系统目录（`src` 内）

### `src/browser/`

浏览器桥、daemon 客户端、标签页、页面/CDP、DOM 快照与辅助等。


| 文件                 | 文件              | 文件               |
| ------------------ | --------------- | ---------------- |
| `base-page.ts`     | `bridge.ts`     | `cdp.ts`         |
| `daemon-client.ts` | `discover.ts`   | `dom-helpers.ts` |
| `dom-snapshot.ts`  | `errors.ts`     | `index.ts`       |
| `page.ts`          | `stealth.ts`    | `tabs.ts`        |
| `utils.ts`         | 及对应 `*.test.ts` |                  |


### `src/pipeline/`

YAML 流水线：模板、执行器、步骤注册与各 **step** 实现。


| 路径                   | 说明     |
| -------------------- | ------ |
| `index.ts`           | 对外导出   |
| `executor.ts`        | 流水线执行  |
| `template.ts`        | 模板与表达式 |
| `registry.ts`        | 步骤注册   |
| `transform.test.ts`  | 转换测试   |
| `steps/browser.ts`   | 浏览器步骤  |
| `steps/download.ts`  | 下载步骤   |
| `steps/fetch.ts`     | 请求步骤   |
| `steps/intercept.ts` | 拦截步骤   |
| `steps/tap.ts`       | tap 步骤 |
| `steps/transform.ts` | 数据变换步骤 |


### `src/download/`

`article-download.ts`、`media-download.ts`、`progress.ts`、`index.ts` 及测试。

### `src/commands/`

内置命令实现，例如 `daemon.ts` 及测试。

### `src/scripts/`

与 CLI 内脚本框架相关：`framework.ts`、`interact.ts`、`store.ts`。

## 九、文档与技能包

- `**docs/`**：面向用户的 VitePress 站点（安装、插件、浏览器桥、adapter 索引、开发者文档等）。
- `**skills/`**：供 Agent 使用的 opencli 技能说明（usage、explorer、oneshot），**非** CLI 运行时依赖。

## 十、维护说明

- 新增站点适配器时，在 `src/clis/<站点>/` 下增加目录，并视需要更新本文档第三节表格。
- 若顶层目录有重大调整（例如新增 `packages/`），请同步更新第二节树与第一节表。
